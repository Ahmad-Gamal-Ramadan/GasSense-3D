## GasSense 3D – Gas Sensor Response Simulator

GasSense 3D is a scientific simulation tool for modeling gas sensor behavior under dynamic experimental conditions.

The application simulates realistic gas sensing responses including ON/OFF cycles, adsorption/desorption kinetics, and environmental effects such as temperature and humidity.

It features an interactive GUI with real-time plots and a live 3D visualization of the sensing process.

### 🔬 Key Features
- Multi-cycle gas exposure (ON/OFF dynamics)
- Physics-inspired adsorption/desorption model
- Realistic response curves similar to research papers
- Adjustable parameters (gas type, concentration, temperature, humidity)
- Live 3D visualization of sensor surface and gas particles
- Time-dependent animation of sensor behavior
- Multiple plots:
  - Gas concentration vs time
  - Sensor response vs time
  - Surface coverage vs time
  - Resistance vs time
- CSV export for further analysis

### 🧠 Technologies
- Python
- PySide6 (GUI)
- PyVista / VTK (3D Visualization)
- NumPy / SciPy (Simulation)
- PyQtGraph (Real-time plotting)

### 🎯 Use Cases
- Educational simulations for nanoscience and sensors
- Research prototyping
- Visualization of gas sensing mechanisms
- Teaching adsorption/desorption dynamics
